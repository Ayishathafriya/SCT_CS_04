#  Keylogger Demonstration 
##  Overview
This repository contains a simple Python script that demonstrates how keystroke logging works using the `pynput` library. The purpose of this project is **educational only**: to help students and cybersecurity enthusiasts understand how attackers may attempt to capture input, and to highlight the importance of detection and prevention.
